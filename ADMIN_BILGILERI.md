# 🔐 Admin Giriş Bilgileri

**Şifre:** `TSR2026.1`

## Vercel'de Environment Variables

Vercel'de bu bilgileri değiştirmek için:

### ADMIN_EMAIL
- **Key:** `ADMIN_EMAIL`
- **Value:** `admin@tsremlak.com` (veya istediğiniz email)

### ADMIN_PASSWORD
- **Key:** `ADMIN_PASSWORD`
- **Value:** `TSR2026.1` (veya istediğiniz şifre)

## ⚠️ Güvenlik Notu

Production'da mutlaka:
1. Güçlü bir şifre kullanın
2. Environment variable'ları ayarlayın
3. Varsayılan şifreyi değiştirin
